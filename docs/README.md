# Docs

This folder contains project documentation for **TruPharma**.

TruPharma is a GenAI-driven drug safety verification system that reconciles
structured regulatory labeling with unstructured post-market adverse event reports.

## What will go here
- system architecture notes and diagrams
- governance and safety rules (refusal policy, grounding/citation requirements)
- prompt design notes and version history
- weekly updates / progress logs
- links to related work and references
